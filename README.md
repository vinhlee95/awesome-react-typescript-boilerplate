## Prerequisites

---

- [NodeJS](https://nodejs.org/en/).
- IDE of your choice.
- Command Line Tools.

## Getting started

---

Install dependencies

```bash
  npm install
```

Start development server

```bash
  npm start
```

## Testing

---

Run test suites:

```bash
  npm run test
```

In watch mode:

```bash
  npm run test:watch
```

## Useful scripts

---

_**Format code and write based on Prettier rules**_

```bash
  npm run format
```

_**Analyse Webpack bundles**_

Development bundle

```bash
  npm run analyze:dev
```

Production bundle

```bash
  npm run analyze:prod
```
